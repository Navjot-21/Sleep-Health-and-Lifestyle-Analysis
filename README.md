# Sleep-Health-and-Lifestyle-Analysis

# Impact of Sleep on Lifestyle and Health

##  Project Overview
This project analyzes the relationship between sleep patterns and various lifestyle and health factors. Using exploratory data analysis (EDA) and data visualization techniques, the study investigates how sleep duration and sleep quality affect stress levels, physical activity, BMI, blood pressure, occupation, and sleep disorders.

---

##  Objective
The objective of this project is to:

- Analyze sleep duration and quality among individuals.
- Study the impact of sleep on physical and mental health.
- Identify patterns related to stress levels, BMI, physical activity, and sleep disorders.
- Gain insights through statistical analysis and visualizations.

---

##  Dataset
The dataset contains information about:

- Gender
- Age
- Occupation
- Sleep Duration
- Quality of Sleep
- Physical Activity Level
- Stress Level
- BMI Category
- Blood Pressure
- Heart Rate
- Daily Steps
- Sleep Disorder

---

##  Technologies Used

- **Python**
- **Pandas** – Data manipulation and analysis
- **NumPy** – Numerical computations
- **Matplotlib** – Data visualization
- **Seaborn** – Statistical graphics
- **Jupyter Notebook** – Interactive analysis

---

##  Project Workflow

### 1. Data Loading
- Imported dataset using Pandas.
- Examined dataset dimensions and structure.

### 2. Data Cleaning
- Checked for missing values.
- Removed duplicates.
- Ensured proper data types.

### 3. Exploratory Data Analysis (EDA)
- Statistical summary of variables.
- Distribution analysis.
- Relationship between sleep and health indicators.

### 4. Data Visualization
Created various plots including:

- Histograms
- Boxplots
- Count plots
- Scatter plots
- Bar charts
- Heatmaps
- Facet grids
- Regression plots

### 5. Interpretation
Insights were drawn to understand how sleep patterns influence overall health and lifestyle.

---

##  Key Insights

- Sleep duration and sleep quality have a strong influence on stress levels.
- Physical activity contributes positively to better sleep quality.
- Individuals with higher stress levels tend to experience poorer sleep.
- BMI and blood pressure show associations with sleep habits.
- Sleep disorders are more common among people with unhealthy lifestyle patterns.
